CSS

Como hemos mencionado, el CSS es un lenguaje informático que especifica
cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan
etc. Un documento suele ser un archivo de texto estructurado con un lenguaje de
marcado: HTML es el más común, pero también existen otros como SVG o XML..
